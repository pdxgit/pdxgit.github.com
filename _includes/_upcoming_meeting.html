The next git together is on!

View on Calagator

PDX Git Together is hosted on the last Wednesday of every month by the fine folks at Elemental Technologies. The format is three twenty minute talks, plus time for questions and socializing.

If you would like to present at the next PDX Git Together, please post to the mailing list.

Talks for the next meeting (2013-03-27) include:

  • The GitHub API, by Christopher Swenson. He will talk a little about what the GitHub API is, how to use it, what it can do, secret undocumented APIs and how to find them, and fun things you can use it for. Christopher will show some examples in Python and Ruby. This talk assumes a basic familiarity with git and GitHub (e.g., the concepts of pull requests and commits).
